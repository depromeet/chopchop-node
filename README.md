# chopchop-node 

### 환경 설정

1. `Node.js`설치 https://nodejs.org/en/
2. 터미널에서 프로젝트 디렉토리 이동
3. `npm install` 실행 (pull, 및 모듈 추가시마다)
4. `npm start` 실행


### or

1. `Node.js`설치 https://nodejs.org/en/
2. 터미널에서 프로젝트 디렉토리 이동
3. `npm install` 실행 (pull, 및 모듈 추가시마다)
4. `npm install -g nodemon` 실행
5. `nodemon ./bin/www` 실행
